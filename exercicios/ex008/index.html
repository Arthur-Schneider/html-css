Principais formatações

Negritos / Destaques

Nesta frase temos um termo em negrito usando a tag <b> (sem semântica)

Nesta frase, temos um termo em destaque usando a tag <strong> (semântica)

Itálico / Ênfase

Nesta frase, temos um texto em itálico usando a tag <i> (não semântica)

Nesta frase, temos um texto em ênfase usando a tag <em> (semântico)

Texto marcado

Podemos criar também um texto marcado usando a tag <mark>

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno neste parágrafo

Texto deletado

Podemos marcar um texto como excluido para indicar que ele deve ser lido, mas não considerado com a tag <del>

Texto inserido

Podemos marcar um texto como inserido para dar ênfase e indicar que eçe foi adicionado depois com a tag <ins>

Existe também o sublinhado com a tag <u> (não semântica)

Texto sobrescrito

Coisas do tipo X2+X+C=0 com a tag <sup>

Texto subescrito

Coisas do tipo H2O com a tag <sub>
